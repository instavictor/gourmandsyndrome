I’ve been here a few times already, so this is now a proper post for it. This is a Jazzy restaurant that you can dine-in without paying a cover which is nice. The jazz itself is pretty fun, although predictable if you don’t listen to a special guest. This is a blackberry gimlet which has reyka vodka, fresh blackberries, and fresh lime. It was pretty good, and it wasn’t too sweet.
